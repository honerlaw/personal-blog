Wave started as a weekend project when I got frustrated with bloated task management apps. I wanted something that lived in my terminal and stayed out of my way.

## Key Features

- Sub-millisecond response times
- Fuzzy search across all tasks
- Git-like syntax for common operations
- Syncs via plain text files (works with Dropbox, iCloud, etc.)

## What I Learned

Building a CLI taught me to obsess over small details. Every keystroke matters. Every millisecond of latency is noticeable. It changed how I think about UX in all my projects.
